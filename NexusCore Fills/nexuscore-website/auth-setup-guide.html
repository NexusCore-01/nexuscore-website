Authentication Setup Guide

Step 1: Set up Auth0 Account

1. Go to auth0.com and sign up for free

2. Create a new tenant for your application

3. Create a new Application (Single Page Application)

4. Copy your Domain and Client ID

Step 2: Set up Formspree for Email Notifications

1. Go to formspree.io

2. Create a free account

3. Create a new form and copy the form ID

Step 3: Update Configuration

Replace in the script section:

const auth0Config = { domain: "your-domain.auth0.com", // ← Replace with your Auth0 domain clientId: "your-client-id", // ← Replace with your Auth0 client ID redirectUri: window.location.origin };

And in the sendSignupNotification function:

fetch('https://formspree.io/f/your-form-id', {

Step 4: Configure Auth0 Settings

In your Auth0 dashboard:

• Add http://localhost:3000 to Allowed Callback URLs

• Add http://localhost:3000 to Allowed Logout URLs

• Add your production domain when you deploy
